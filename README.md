# Local Intelligence Brief

Daily automated news digest for El Sauzal, Baja California, Mexico.

## Setup
1. Copy `.env.example` to `.env` and add your API keys
2. Install: `pip install -r requirements.txt`
3. Run: `python test_run.py`

## API Keys needed
- `ANTHROPIC_API_KEY` — for Claude LLM analysis
- `NEWSAPI_KEY` — free tier at newsapi.org
- `TELEGRAM_BOT_TOKEN` — optional, for Telegram delivery

## GitHub Pages
Push to GitHub, then enable Pages: Settings → Pages → Source: `main` / `/docs`
